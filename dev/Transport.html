Class: Transport

Transport(showLevel)

new Transport(showLevel)

Ways to analyze log events. Each transport does something with log event: prints to console, store in file, etc. See also Logger.

Parameters:
Name Type Default Description
showLevel string info

If level is equal or higher than the value it will be analyzed. Possible values: 'debug', 'info', 'warn', 'error', 'crit'
Properties:
Name Type Description
showLevelNum number

Numeric identifier of showLevel value: 0, 1, 2, 3, 4.

Methods

analyzer()

Actions to perform when call log in parent Logger.
