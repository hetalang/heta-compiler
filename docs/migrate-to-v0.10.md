# Migration from v0.9 to v0.10

The v0.10 release of Heta compiler includes updates to support the latest Julia 1.12, includes new export format "Canonical", and adds support for the `hasMeta` action.

The **v0.10** version of the Heta compiler adheres to the [Heta standard](/specifications/) **v0.5.2**.

No special updates to the modeling platforms are required. Just update `builderVersion: ^0.10.0` in the platform declaration file.
